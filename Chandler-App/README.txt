==========================
 The Chandler Application
==========================

This project is currently a catch-all for anything that isn't the platform or
a plugin.  As the packages incubated here get bigger, they'll split off into
their own projects, leaving this as the home for the "main" application
structure, and most of the app-specific presentation and interaction code.
